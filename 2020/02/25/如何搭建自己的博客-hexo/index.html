如何搭建自己的博客(hexo)

2020-02-25

咋样搭建自己的个人博客

第一步:下载node.js //下载地址:https://nodejs.org/en/
第二步:(win)win+r进入命令行测试 node -v npm -v是否正确
第三步:借助包管理器淘宝的cnpm(npm install -g cnpm –registry=https://registry.npm.taobao.org）
第四步:测试cnpm cnpm -v 是否正确
第五步:创建hexo框架 cnpm install -g hexo-cli 并测试 hexo -v 是否正确
第六步:测试路径:chdir 并创建文件夹 mkdir blog 并进入文件夹中 cd blog/

第七步:hexo init进行hexo的创建博客,并查看当下目录dir

第八步:hexo s 查看是否生成hexo网站:4r，如果要断开ctrl+c即可

第九步:复制到浏览器上

第十步:创建博客: hexo n “我的第一篇博客文章”，会生成一个目录

第十一步：进入cd source/_posts/这个目录下，dir查看当前目录

第十二步:我的第一篇博客文章.md进行编写即可

第十三步:cd ../..退回blog目录下

第十四步:hexo clean 清理博客 hexo g 生成博客 在进行hexo s一下

第十五步:进入github上部署一个仓库 名字: 名字.github.io 描述:我的hexo博客

第十六步:装一个git的部署插件:cnpm install –save hexo-deployer-git

第十七步:_config.yml打开文件进行修改 Deployment下

type: git

repo: https://github.com/Engineer-Programmer/Engineer-Programmer.github.io.git

branch: master

第十八步:打开github.com刷新页面复制:Engineer-Programmer.github.io并记住前面是https:协议

或者修改git命令上的user.name ,user.email

先修改git init

git config –replace-all user.name “github的用户名”

git config –replace-all user.email “qq邮箱”

第十九步:https://github.com/litten/hexo-theme-yilia进入自定义github hexo页面后

然后进行克隆命令行:git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia

第二十步:修改Extensions下的theme: yilia

第二十一步:hexo clean 清理博客 hexo g 生成博客 在进行hexo s一下:在把4000端口复制到网站上即可

第二十二步:hexo d 传到远端 再刷新一下hexo网页即可 ,Engineer-Programmer.github.io即可进入生成的网页

https://hexo.io/themes/ :hexo的界面
